# sexprs

A minimal lisp dialect to evaluate written entirely in rust.

```
cargo install sexprs
sexprs
```

> Nomenclature comes from "s-exprs", that is: [**S-EXPR**essions](https://en.wikipedia.org/wiki/S-expression)


## Try the REPL

```bash
cargo run
```

or

```bash
cd crates/repl
cargo run
```


## Run tests

```bash
cargo test --all
```
